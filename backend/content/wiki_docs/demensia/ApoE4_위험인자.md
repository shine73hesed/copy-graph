# ApoE4 유전자 위험인자 (ApoE4 Risk Factor)

## 핵심 개념

## 4. 위험인자

### 4-1. 비수정 가능 위험인자

| 위험인자 | 상세 |
|--------|------|
| 고령 | 65세 이상에서 5세마다 유병률 2배 증가 |
| 여성 | 유병률 남성의 약 2배. 뇌혈관장벽이 남성보다 약함 |
| ApoE4 유전자 | E4 보유 시 치매 위험 3~15배. 혈관주위세포가 ApoE 과다 생성 → Aβ 응집 유도 |
| 가족력 | 1차 친척 치매 시 위험 2~4배 |

## 4. 위험인자

### 4-1. 비수정 가능 위험인자

| 위험인자 | 상세 |
|--------|------|
| 고령 | 65세 이상에서 5세마다 유병률 2배 증가 |
| 여성 | 유병률 남성의 약 2배. 뇌혈관장벽이 남성보다 약함 |
| ApoE4 유전자 | E4 보유 시 치매 위험 3~15배. 혈관주위세포가 ApoE 과다 생성 → Aβ 응집 유도 |
| 가족력 | 1차 친척 치매 시 위험 2~4배 |
| 두부외상 | 만성 외상 후 Aβ 축적 가속화 |

ApoE4 유전자와 관련하여 Blanchard et al.(2020)은 ApoE4가 있으면 혈관주위세포(pericytes)가 ApoE 단백질을 과다 생성하고, 이것이 Aβ의 응집을 유도하며 특히 뇌혈관 벽에 아밀로이드가 쌓여 뇌혈관장벽(Blood Brain Barrier)의 기능을 훼손한다고 밝혔다.

### 4-2. 수정 가능 위험인자

| 위험인자 | 기전 |
|--------|------|
| 고혈압 | 뇌혈관 손상 → 혈관성 치매 + 혼합형 위험 증가 |
| 당뇨 | 과도한 당 대사 → 알츠하이머 위험 단백질 증가 |
| 비만 | 60대 과체중 → 대뇌피질(회백질) 위축 가속화. 내장 지방 → 뇌 염증 유발 |
| 고콜레스테롤 | 뇌혈관 동맥경화 → 뇌혈류 저하 |
| 흡연·음주 | 대뇌 피질(회색질+백질+뇌용적) 감소 가속화 |
| 운동 부족 | 신체 활동 부족이 알츠하이머 원인의 약 13% |
| 고염분 식이 | 소듐 농도 증가 → 시상하부 뉴런 활성 → 혈류 감소 → 저산소증 |
| 저학력 | 인지 예비력(cognitive reserve) 감소 |
| 사회적 고립 | 인지 자극 부족, 우울 위험 증가 |

### 4-3. 치매 전조 증상으로서의 후각 저하

노년기 후각 기능 저하는 치매의 전조 증상일 수 있다. Son et al.(2021)은 알츠하이머병 초기에 Aβ가 후각신경계 영역, 특히 외측 비갑개부터 배쪽 후각구 영역에 많이 축적되며, 후각신경세포의 퇴화와 재생의 불균형이 치매 초기부터 나타난다고 보고하였다.
