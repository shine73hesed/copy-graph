### Chapter 3. 환각 (Hallucination)

#### 정의
- 적절한 외부 자극이 없는 상태에서 발생하는 감각적 경험
- 환각 vs 착각(illusion): 착각은 실제 자극을 왜곡, 환각은 자극 없이 경험
- 위환각(pseudohallucination): 환자가 각성상태에서 체험하지만 실제가 아님을 인식

#### 분류
| 유형 | 특징 |
|-----|-----|
| **환시(Visual)** | 가장 흔함 (특히 DLB). 단순환시(빛, 색깔, 기하학적 모양) vs 복합환시(사람, 동물) |
| **환청(Auditory)** | 조현병에서 가장 특징적. 사람 목소리, 음악 등 |
| **환촉(Tactile/Haptic)** | 벌레가 기어가는 느낌(formication), 콕콕 찌르는 느낌 |
| **환후(Olfactory)** | 측두엽전증에서 비교적 흔함 |
| **환미(Gustatory)** | 드뭄 |
| **신체환각(Somatic)** | 장이 뒤집히는 느낌, 심장이 가려운 느낌 등 |

#### 케이스 (63세 남성)
- 기억력 장애 + 환청 + 환시 + 망상
- 이웃집에서 감시카메라로 나를 지켜본다, 벽에서 소리가 들린다
- MMSE 15점, 기억 회상 0점(0/3)
- 보호자: 부부가 따라와 방 안까지 들어온다고 말씀

#### 환각 원인
- 조현병, 정신병, 알츠하이머병, 뇌전증, 뇌졸중, 다발성 경화증, 헌팅턴병, 윌슨병
- 약물, 수면부족, 영양결핍, 알코올중독
- **알츠하이머병**: 환각 유병률 12~49% (연구마다 3~67%)
  - 환시 15%, 환청 12%
  - 중증도 증가 시 환시·환청 모두 증가
  - DLB 대비경쇄치매(subcortical limbic structures)의 레비소체에서 40~75% 환시
- **루이소체치매**: 환시가 가장 흔하고 진단에 필수적
  - 구체적 형태(동작, 색깔 포함)로 생생하게 표현
  - 신경이완제과민반응(neuroleptic sensitivity reaction)

#### 생물학적 연관성
- 해마곁이랑(parahippocampal gyrus) 신경세포 증가
- 배봉솔기핵(dorsal raphe nucleus) 신경세포 감소
- 루이소체치매: 5-hydroxytryptamine 보존, 콜린(choline) 감소
- 환시 발생: 일차시각피질(primary visual cortex) → 시각연합영역(visual association cortex) 경로
- 시각계(visual system) 구심로차단(deafferentation) → 피질방출현상(cortical release phenomenon)

#### 치료
- 효과가 입증된 치료 많지 않음
- 항정신병약물 또는 비정형항정신병약 주의깊게 사용
- 환각 유발 약물 중지
- 스트레스 관리, 충분한 수면
