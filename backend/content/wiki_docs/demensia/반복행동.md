# 반복행동 (Repetitive Behaviors)

## 핵심 개념

#### 피질-선조체 회로 (Corticostriatal Circuitry)
- 인간의 반복행동은 목표 지향적 행동, 시각적·청각적·체성감각 자극을 거치고 수행
- **피질-선조체 회로(corticostriatal circuitry)** 이상이 반복행동의 핵심
- 4가지 루프:
  - **감각운동 루프(sensorimotor loop)**: 해당 부분 도파민 농도 이상 → 상동행동(stereotypy)
  - **연합 루프(associative loop)**: 전두엽 관련 영역과 관련, 인지적 유연성(cognitive flexibility) 저하
  - **변연계 루프(limbic loop)**: 정서적 도파민 조절 → 이상되면 충동적 반복행동
  - **강박 루프**: 안와전두피질(orbitofrontal cortex) + 전측대상피질(anterior cingulate) + 복측 선조체(ventral striatum)
- 그림 4: 피질-선조체 회로 도식 (Cortex → Striatum → Pallidum → Thalamus → Cortex)

### 전두측두엽치매 환자에서의 반복행동
- 전두측두엽치매(FTD) 반복행동 유병률 약 60~87%로 매우 흔함
- 알츠하이머 대비 구어 반복, 요구의 반복, 의미 없는 행동 반복 등이 현저
- 단순 반복: 두드리기, 문 여닫기, 또 열기, 모으기, 걸어 다니기, 도시락 싸기
- 복합 반복: 전화걸기, 같은 질문, 정해진 시각에 반복적 외출
- FTD에서 반복행동은 전두엽(안와전두피질/배외측전전두엽)과 선조체(꼬리핵/caudate) 손상과 관련
- 의미치매(semantic dementia)에서도 비슷한 반복행동 관찰

### 알츠하이머병 환자에서의 반복행동
- 유병률 약 60~87%로 FTD와 비슷한 수준
- MMSE 점수가 낮을수록, 인지적 유연성 저하 시 반복행동 더 심함
- set shift(전환능력) 이상이 핵심
- 반복행동은 인지적 측면과 밀접한 연관 있으나 도파민 저하도 영향

### 반복행동의 치료
- 환경적 중재 우선: TV 포인트이, 기계 비디오, 보호자 비디오 시청
- 약물: 세로토닌 계통 약물 시도 가능하나 근거 부족
- 산책로프로그램, 마시지테라피, 감각자극활동테라피 등
- 비약물적 치료 우선, 약물 최소화
