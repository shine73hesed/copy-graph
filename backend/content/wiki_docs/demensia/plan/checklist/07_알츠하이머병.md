튜터는 아래 5개 항목이 학습자에 의해 모두 다뤄진 후에만
[TOPIC_COMPLETE]를 발행할 수 있습니다.
순서는 학습자의 반응에 따라 자유롭게 결정하세요.
학습자가 이미 아는 항목은 확인만 하고 넘어가세요.

□ C1: 20년 카운트다운 — 증상 발현 15~20년 전부터 뇌에서 병리적 변화가 시작. 50대에 시한폭탄 째깍거리는데 70대에야 증상 드러남. 전임상 단계의 존재
□ C2: 방아쇠와 총알 — 아밀로이드 플라크(방아쇠, 쓰레기 축적)와 타우 단백질(총알, 미세소관 붕괴→신경섬유매듭). 방아쇠 당겨져도 총알이 반드시 발사되지는 않음(정상 노인도 플라크 존재). 타우 확산이 실제 중증도 결정
□ C3: 신경염증과 연쇄 붕괴 — 미세아교세포/별아교세포가 폭주→과산화수소 분출(커피 얼룩 지우려고 표백제로 집 전체를 녹이는 격). LRP1 수용체를 통해 독성 타우가 전염병처럼 뇌 전체로 확산
□ C4: 아세틸콜린 공장 파괴와 약물 원리 — 마이네르트 기저핵(독점 공장) 파괴→아세틸콜린 고갈. 도네페질(콜린에스터라제 억제제)=남은 제품의 분해를 막아 조금이라도 더 오래 쓰게. 항콜린제 동시 투여 금기(살리려는 것과 죽이려는 것의 충돌)
□ C5: 뇌 동네별 타격 순서와 사례 적용 — 해마(1차, 단기기억)→전두엽(2차, 판단·브레이크)→두정엽(시공간)→SCN(생체시계). Mr Ho 사례에서 증상별 파괴 부위 매핑

체크 규칙:
- 학습자가 해당 개념을 "자기 말로" 표현하면 체크
- 교재 문장을 그대로 반복하는 것은 체크 안 함
- 한 턴에 여러 항목이 동시에 체크될 수 있음

## 자료

```clinical
[임상 사례 — Mr Ho (78세 남성)]
12개월간 기억력 저하, 낮과 밤 혼란, 장소 지남력 장애, 배회, 단것 갈망
동반 질환: 뇌졸중, COPD, 만성 신장질환, 고콜레스테롤
검사: MMSE 14/30, CT에서 과경색·만성 허혈 소견
진단: 알츠하이머 + 혈관성 혼합형 치매
치료: 도네페질 5mg + 주간보호
```

```tip
[핵심 비유 모음]
- 시한폭탄: 20년 전부터 째깍거리는데 집주인은 모름
- 방아쇠(아밀로이드)와 총알(타우): 방아쇠 당겨져도 총알이 반드시 발사되지 않음
- 기차 선로: 타우=선로의 나사, 나사 빠지면 기차 탈선
- 표백제: 별아교세포 폭주=커피 얼룩 지우려고 집 전체를 표백제로 녹임
- 공장 파괴: 마이네르트 기저핵=아세틸콜린 독점 공장, 공장 불타면 남은 제품이라도 아껴 쓰기(도네페질)
- 동네별 화재: 해마→전두엽→두정엽→SCN 순서로 불길이 번짐
```
