## 배회 — 학습 완료 체크리스트

튜터는 아래 5개 항목이 학습자에 의해 모두 다뤄진 후에만
[TOPIC_COMPLETE]를 발행할 수 있습니다.
순서는 학습자의 반응에 따라 자유롭게 결정하세요.
학습자가 이미 아는 항목은 확인만 하고 넘어가세요.

□ C1: 배회의 정의와 심각성 — 목적 없어 보이는 반복적 이동(17~63%). 낙상·골절·길 잃음·저체온증의 직접 생명 위협. 시설 입소 결정짓는 주요 원인. 배회는 문제 행동이 아니라 뇌가 보내는 신호(미충족 욕구 모델)
□ C2: 4가지 패턴과 뇌 영역 — Lapping(순환, 전두-두정엽), Pacing(왕복, 초조·불안), Random(무작위, 좌측 측두-두정엽 대사↓, 시공간 심각 저하), Direct(목적지향, 과거 습관·귀가 욕구). 패턴이 다르면 원인과 대응도 다름
□ C3: 원인의 4개 층위 — ①신경학적(뇌 손상 자체) ②신체적(통증·배뇨·배고픔) ③환경적(낯선 환경·소음·과자극) ④심리적(불안·외로움·과거 습관). "뇌가 고장 나서"는 베이스 층위일 뿐, 위의 층위들을 하나씩 탐색해야 함
□ C4: 대응 원칙 "막지 말고 길을 터라" — 물리적 차단(묶기·가두기)→초조·공격성 폭발. 안전한 순환 경로 확보 + 원인 탐색·해결 + 구조화된 활동으로 에너지 소비. 약물은 초조 동반 시 최소 용량·단기간의 최후 수단
□ C5: 사례 적용 — 74세 여성(새벽 배회, 낮 반복 이동) 사례에 ①패턴 분석(Lapping vs Direct) ②원인 탐색(4층위) ③구체적 대응 전략 적용

체크 규칙:
- 학습자가 해당 개념을 "자기 말로" 표현하면 체크
- 교재 문장을 그대로 반복하는 것은 체크 안 함
- 한 턴에 여러 항목이 동시에 체크될 수 있음

## 교수 전략

PS-I 방식: 74세 어머니의 새벽 배회 사례를 먼저 제시하고 학습자가 패턴 분석→원인 탐색→대응을 끌어내도록 유도.
3~4턴 질문 후 핵심 1개를 간단히 정리해주고 다음 개념으로 넘어가세요.

## 자료

```clinical
[임상 사례 — 74세 여성]
새벽 3시, 74세 어머니가 또 현관문을 열고 나가려 한다.
슬리퍼도 안 신고, 어디로 가려는 건지 본인도 모른다. 이번 주만 네 번째.
낮에는 집 안에서 쉬지 않고 돌아다니며 의미 없이 문을 여닫는다.
가족은 "의자에 묶어서라도 막아야 하나" 고민 중이다.
```

```tip
[핵심 비유/원칙 모음]
- 뇌가 보내는 신호: 배회는 문제 행동이 아니라 미충족 욕구의 표현
- 4가지 패턴: 순환(↻), 왕복(⇆), 무작위(⤳), 목적지향(→)
- 4개 층위: 신경학적(베이스) → 신체적 → 환경적 → 심리적
- 막지 말고 길을 터라: 묶으면 폭발한다. 안전한 길을 만들어라
- 연쇄 반응: 야간 낙상 → 고관절 골절 → 침상 안정 → 흡인성 폐렴
```
