## Chapter 7. 경도행동장애 (Mild Behavioral Impairment, MBI)

### 케이스 (67세 여성)
- 총명한 교수 출신, 최근 기억감소와 의욕 감퇴
- 사소한 물건 자주 찾음, 약속 시간 혼동, 이전에 즐기던 활동에 무관심
- K-MMSE 27점(recall -3), GDS 3, CDR 0.5(sum of box 3.0), K-ADL 0.5
- 기억 저하 + 동기감소 + 성격변화 → amnestic MCI인가 MBI인가?

### 배경
- **MBI(Mild Behavioral Impairment)**: 인지기능이 아니라 행동/심리증상이 먼저 두드러지는 치매 전 단계
- SCD(주관적인지저하) → MCI(경도인지장애) 경로와 별도로, MBI 경로 존재
- MBI는 NPS(neuropsychiatric symptoms)가 치매의 prodromal 단계에서 나타남을 설명하는 진단적 구성개념(diagnostic construct)

### 인지기능의 연속체에서 MBI의 위치
```
정상(Normal) → SCD → MCI → Dementia
               ↓
              MBI → Dementia
```
- 그림 1: Cognitive and behavioral pre-dementia risk axes
  - 인지 축: Normal → SCD → MCI(경도인지장애) → Dementia
  - 행동 축: Normal → MBI(경도행동장애) → Dementia
  - 인지 축에는 MMSE, MoCA, CDR 등
  - 행동 축에는 MBI-C(MBI Checklist), NPI, BPSD-Q 등

### MBI의 진단기준 (ISTAART NPS-PIA criteria)
- **ISTAART**(International Society to Advance Alzheimer's Research and Treatment)의 NPS Professional Interest Area에서 제시한 진단 기준:
  1. 50세 이상 행동/성격 변화가 6개월 이상 지속
  2. 환자의 장기적 기본 성격과 뚜렷한 차이
  3. 공식적 인지검사에서 치매 수준 아님(정상 또는 MCI)
  4. NIA-AA consensus criteria의 치매 진단에 해당하지 않음
  5. 행동 변화가 다른 정신질환으로 설명되지 않음

### MBI Checklist (MBI-C)
- 5개 MBI 도메인별 예측 조사 항목:

| 도메인 | 설명 |
|------|-----|
| **동기감소(Decreased motivation, drive)** | 의욕 저하, 관심 감소, 사회적 참여 감소, 일상활동 무관심 |
| **정서조절장애(Affective dysregulation)** | 불안, 우울, 슬픔, 쾌락무능(anhedonia), 공황발작 |
| **충동조절장애(Impulse dyscontrol)** | 초조, 공격성, 충동적 행동, 도박, 무분별한 소비, 부적절한 성적 행동 |
| **사회적 부적절성(Social inappropriateness)** | 공감능력 상실, 사회적 규범 무시, 무례한 언행, 부적절한 발언 |
| **비정상적 지각/사고(Abnormal perception/thought content)** | 환시, 환청, 망상, 의심, 편집증 |

### 관련 질환과의 감별
- **행동변이형 전두측두엽치매(bvFTD)** 진단기준(International FTD Criteria Consortium)과 일부 중첩
  - 조기 탈억제, 무감동, 공감능력 상실, 강박, 과식, 사회적 규범 위반
  - bvFTD는 MBI보다 심한 단계
- Transdiagnostic criteria: MBI는 진단 확정 전 단계의 치매 조기 경보

### 진단 고려사항
- ISTAART 진단 기준 사용 시 **minimal functional impairment** 수준에서 기능저하가 미미해야 함
- 기능 저하가 인지저하에 의한 것인지 행동증상에 의한 것인지 구분 필요
- ISTAART NPS-PIA consensus criteria 와 NIA-AA consensus criteria 모두 충족 여부 확인

### 결론
- MBI는 치매의 행동적 전구기(prodrome)를 조기 포착하기 위한 개념
- 치매 예방적 관점에서 MBI 선별이 중요
- NPI(신경정신행동검사) 이외에 MBI-C 같은 특화 도구 활용 필요
- 인지 축과 행동 축 모두 고려한 조기 개입 전략 필요
